# Agente Pulse 14

Base limpia v2.

Primer paso del agente:

- Subir una o varias imágenes.
- Guardarlas en una biblioteca de trabajo.
- Asignar un porcentaje numérico a cada imagen.
- No permitir que la suma supere el 100%.
- Validar en servidor que la suma sea exactamente 100% antes de continuar.
