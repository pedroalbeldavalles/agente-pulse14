# Módulo 1 - Preparación IA

Estado: validado como base funcional.

Este módulo prepara las imágenes de referencia y sus porcentajes para enviarlas al Motor IA Pulse14.

## Reglas validadas

- Las imágenes pueden ser múltiples.
- Al cargar imágenes, el sistema reparte automáticamente el 100% entre ellas.
- El usuario puede modificar cada porcentaje de forma numérica.
- La suma no puede superar el 100%.
- El botón **Crear imagen con IA** se activa solo cuando la suma es 100%.
- Solo debe existir un botón principal de creación IA.
